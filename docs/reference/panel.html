Panel

Constructor

Panel( options : Object )


- options

  • id: Id of the element
  • className: Add class to the element
  • width: Width of the panel DOM element [fit space]
  • height: Height of the panel DOM element [fit space]
  • style: CSS Style object to be applied to the panel

Properties


.branches : Array

Array of opened branches in the panel

.onevent : Function

Function called when any of its widgets changes its value

.root : HTMLElement

Container element of the panel (div)

.widgets : Object

Data of all the widgets created inside the panel

Methods


.attach ( content : HTMLElement )

Appends a child to the panel root. If "content" has a root, then appends it, if not, append "content".

.branch ( name : String, options : Object ) : Branch

Open new closable branch (section) inside the panel.

.clear ( )

Clear the panel root and all of its widgets.

.clearQueue ( )

Removes the queued container, so widgets will be appended in its corresponding branch/container.

.endLine ( )

Stop inlining widgets when created.

.get ( name : String ) : Widget

Returns a given widget (in the panel) searched by name.

.getBranch ( name : String ) : Branch

Returns a given branch (in the panel) searched by name.

.getValue ( name : String ) : *

Returns the value of a given widget (in the panel) searched by name.

.merge ( )

Merges current opened branch, meaning the following widgets will be created directly to the panel.

.queue ( container : HTMLElement )

Sets a new temporal container for the newly created widgets. To stop using it, call Panel.clearQueue.

.sameLine ( n : Number )

Sets the following N widgets to be inlined when created. If N is null, every widget created will be inlined until Panel.endLine is called.

.setValue ( name : String, value : * )

Sets the value of a given widget (in the panel) searched by name.
