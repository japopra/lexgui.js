Tabs

Constructor

Tabs( area : Area, options : Object )


- options

  • fit: Style of the tabs. Use all parent width.
  • folding: Tabs can be folded/minimized
  • onclose: Function to be called when closing tabs.

Properties


.area : Area

Area of the tab container.

.root : HTMLElement

Container element of the tabs DOM element (div)

.selected : String

Name of the selected tab if any

.tabs : Object

Map containing information for each of the generated tabs.

.tabDOMs : Object

Map containing the DOM elements for each of the generated tabs.

Methods


.add ( name : String, content : HTMLElement, options : Object )

Adds a new tab inside the tab container. As options, you can use:

  • selected: If the tab is selected by default.
  • onCreate: Function to be called when creating the tab DOM element.
  • onSelect: Function to be called when selecting the tab.

.select ( name : String )

Selects a tab, calling onSelect option callback if used when the tab was added.

.delete ( name : String )

Deletes permanently the tab.
