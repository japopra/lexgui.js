Area

Constructor

Area( options : Object )


- options

  • id: Id of the element
  • className: Add class to the element
  • width: Width of the area element [fit space]
  • height: Height of the area element [fit space]
  • skipAppend: Create but not append to GUI root [false]

Properties


.root : HTMLElement

Container element of the area (div)

.sections : Array

If split, the two split areas

Methods


.addMenubar ( callback : Function, options : Object ) : Menubar

Creates a Menubar inside the area using the callback to create each of the entries and subentries.

.addOverlayButtons ( buttons : Array, options : Object )

Creates a group of buttons that can be positioned at specific area locations. options.float = "htc" where h: horizontal; v: vertical; t: top; m: middle; b: bottom; l: left; c: center; r: right

.addPanel ( options : Object ) : Panel

Creates a Panel and attaches automatically to the area.

.addTabs ( options : Object ) : Tabs

Creates a tab section inside the area. Use Tabs.add() to create and append content for each of the tabs.

.attach ( content : HTMLElement )

Appends a child to the area (e.g. a Panel or any HTMLElement). If "content" has a root, then appends it, if not, append "content".

.propagateEvent ( eventName : String )

Propagates an event recursively through all the inner sections of the area in the form area.eventName(boundingRect)

.setSize ( size : Array )

Resize both sections of the area using an Array [width, height].

.split ( options : Object ) : Array

Splits the area into another 2 either vertically or horizontally.
