Code Editor


The Code Editor component supports the following language highlightings:

  • Plain text
  • JavaScript
  • CSS
  • WGSL, GLSL
  • JSON
  • XML

Creating the editor

The Editor has to be appended to an existing area. It means that it can be created anywhere in the document window (e.g. dialogs, panels, areas, etc.), since to create an area it's as easy as instancing a new one. The following lines instanciate the DOM element for the editor: 

let editor = new LX.CodeEditor( area );

When creating it, you can use different options to adapt the editor to your needs:

  • allow_add_scripts (bool): Enables/Disables adding new tabs when double-clicking the tab space and doesn't add the 'New tab' tab.
  • skip_info (bool): Show or hide the editor info panel at the bottom (Line, Column, Change Hightlighting, etc.).
  • name (String): If defined, name of the default code tab.
  • highlight (String): Default Hightlighting
  • onsave (Function): Code to call when pressing Ctrl+S in the editor.
  • onrun (Function): Code to call when pressing Ctrl+Enter in the editor.

Opening files and new tabs

To load existing files you must use CodeEditor.loadFile( filename ). This will open a new tab with the code in it.

Tab Events:

  • Double click to tab space: Adds new empty tab
  • Middle click to tab: Closes tab
